# Catálogo de Tecnologias

Projeto desenvolvido durante uma imersão de desenvolvimento, com o objetivo de praticar HTML, CSS e JavaScript consumindo dados de um arquivo JSON.

## Sobre o projeto

O projeto exibe um catálogo de tecnologias da área de desenvolvimento de software.  
Cada card mostra:
- Nome da tecnologia  
- Descrição  
- Data de criação  
- Tags  
- Link oficial com o botão **"Saiba mais"**

Os dados são carregados dinamicamente a partir do arquivo `data.json` usando JavaScript.

## Tecnologias utilizadas

- HTML5  
- CSS3  
- JavaScript (DOM + fetch de arquivo JSON)  

## Estrutura do projeto

- `index.html` – Estrutura da página  
- `style.css` – Estilização da interface (layout e cards)  
- `script.js` – Lógica para carregar e exibir os dados do `data.json`  
- `data.json` – Lista de tecnologias e suas informações
